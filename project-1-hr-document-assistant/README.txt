# Project 1 — HR Document Intelligence Assistant

## What It Does
RAG-powered HR chatbot built on AWS Bedrock that ingests HR 
policies and answers employee questions with source citations.

## Tech Stack
- AWS Bedrock Managed Knowledge Base
- Amazon Nova Pro (foundation model)
- Bedrock Agent Runtime — Retrieve API
- Python + boto3
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Skills Demonstrated
RAG Pipeline · AWS Bedrock · Amazon S3 · 
Metadata Engineering · Responsible AI
